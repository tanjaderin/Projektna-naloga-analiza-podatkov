Youtuberji
===

Pri projektni nalogi bom analizirala najpoularnejše YouTube kanale, podatke bom vzela na strani :
https://socialblade.com/youtube/top/5000

O vsakem kanalu bom zejela :
* mesto na lestvici
* oceno
* ime 
* število objav
* število ogledov
* število vpisanih
